# Guess to Survive

## Environment

1. Copy `.env.example` to `.env.local`
2. Fill in the values for your environment

Vite exposes only variables prefixed with `VITE_` to the client.

## Development

- `pnpm dev`
- `pnpm build`
- `pnpm preview`

## Code Quality

- `pnpm lint` (Ultracite check)
- `pnpm lint:fix` / `pnpm format` (Ultracite fix)
- `pnpm doctor` (Ultracite doctor)
